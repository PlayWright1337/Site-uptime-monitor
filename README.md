# 🤖 Site Uptime Monitor Bot

Telegram-бот для мониторинга доступности сайтов с мгновенными оповещениями в личные чаты и группы.

---

## ✨ Возможности

- 🔍 **Автоматическая проверка** — периодически пингует все добавленные сайты
- 🚨 **Оповещения о падении** — мгновенно сообщает когда сайт недоступен
- ✅ **Оповещения о восстановлении** — сообщает когда сайт снова заработал (с временем простоя)
- 📊 **Статус-дашборд** — актуальный статус всех сайтов с временем отклика
- ⚙️ **Настройка через бота** — интервал, таймаут и порог оповещений прямо в Telegram
- 👥 **Групповые уведомления** — рассылка в группы и каналы
- 💾 **SQLite база данных** — все сайты и настройки сохраняются между перезапусками

---

## 🚀 Установка

### 1. Клонируйте репозиторий

```bash
git clone [<repo-url>](https://github.com/PlayWright1337/Site-uptime-monitor)
cd site-uptime-monitor
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Настройте `.env`

Скопируйте `.env.example` → `.env` и заполните:

```env
BOT_TOKEN=ваш_токен_от_BotFather

# Ваш Telegram ID (узнайте через @userinfobot)
ADMIN_IDS=123456789

# Дополнительные пользователи для уведомлений (опционально)
USER_IDS=111111111,222222222

# ID групп/каналов для уведомлений (опционально, можно оставить пустым)
GROUP_IDS=-1001234567890,-1009876543210
```

> **Важно**: `GROUP_IDS` можно оставить пустым — бот не упадёт.

### 4. Запустите бота

```bash
# Продакшн
npm start

# Разработка (с авто-перезапуском)
npm run dev
```

## 📱 Интерфейс бота

### Главное меню
```
🌐 Сайты      📊 Статус
⚙️ Настройки  ℹ️ Помощь
```

### Меню сайтов
```
📋 Список сайтов
➕ Добавить сайт
🗑 Удалить сайт
🔄 Проверить сейчас
◀️ Главное меню
```

### Настройки
| Параметр | Значения | По умолчанию |
|---|---|---|
| Интервал проверки | 1, 5, 10, 30, 60 мин | 5 мин |
| Таймаут соединения | 5, 10, 15, 30, 60 сек | 10 сек |
| Порог оповещений | 1, 2, 3, 5 провалов | 1 |

> **Порог оповещений** — сколько провалов подряд нужно для отправки уведомления (защита от ложных срабатываний).

## ⚙️ Роли пользователей

| Переменная | Роль | Может |
|---|---|---|
| `ADMIN_IDS` | Администратор | Управлять ботом + получать уведомления |
| `USER_IDS` | Пользователь | Только получать уведомления |
| `GROUP_IDS` | Группа/канал | Только получать уведомления |

## 🔧 Технологии

- **[Telegraf v4](https://telegraf.js.org/)** — Telegram Bot Framework
- **[node:sqlite](https://nodejs.org/api/sqlite.html)** — Встроенный SQLite (Node.js 22.5+, без компиляции)
- **[axios](https://axios-http.com/)** — HTTP-клиент для пинга сайтов
- **[node-cron](https://github.com/node-cron/node-cron)** — Планировщик задач
- **[dotenv](https://github.com/motdotla/dotenv)** — Загрузка .env
